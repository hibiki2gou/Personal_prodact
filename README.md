# 自動英訳和訳機能 (英訳・和訳ポップアップ拡張機能)

ウェブページ上で選択したテキストを、日本語なら英語へ、それ以外なら日本語へ自動判定して翻訳する Google Chrome 拡張機能です。
翻訳結果をワンクリックでクリップボードにコピーできる機能も搭載しています。

##  主な機能

- **自動言語判定**: 日本語を選択すれば英訳、英語（その他）を選択すれば和訳を自動で行います。
- **ポップアップ表示**: 選択したテキストのすぐそばに翻訳結果が表示されるため、視線を動かさずに内容を確認できます。
- **ワンクリックコピー**: 翻訳結果をボタン一つでコピーでき、ドキュメント作成や学習を効率化します。

##  インストール方法

1. このリポジトリをローカル環境にダウンロード、または `git clone` します。
2. Google Chrome を開き、アドレスバーに `chrome://extensions/` を入力して実行します。
3. 右上の「デベロッパー モード」を **ON** にします。
4. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
5. ダウンロードしたプロジェクトのフォルダ（`manifest.json` が含まれる階層）を選択します。

##  使い方

1. Google Chromeのブラウザで任意のウェブページを開きます（拡張機能を読み込んだ後にページを更新してください）。
2. 翻訳したい文章をマウスでドラッグして選択します。
3. マウスを離すと、選択範囲の近くに翻訳結果のポップアップが表示されます。
4. 翻訳結果の下にある「コピー」ボタンを押すと、結果がクリップボードに保存されます。
5. ポップアップ以外の場所をクリックすると、ポップアップが閉じます。

##  ファイル構成

- `manifest.json`: 拡張機能の設定ファイル。
- `content.js`: 翻訳ロジック、ポップアップ生成、イベント制御を担当。
- `style.css`: ポップアップのレイアウトとアニメーションのデザイン。

##  使用技術

- JavaScript (Vanilla JS / Async-Await)
- CSS3
- Google Translate API (Unofficial via fetch)
